## 자료 추가 방법

웹앱 안에서 강의를 열고 **"자료 업로드"** 버튼을 누르면 해당 GitHub 폴더가 바로 열립니다. 거기서 파일을 드래그&드롭하고 "Commit changes"만 누르면 됩니다. 1~2분 뒤 자동으로 페이지에 나타납니다.

## 동작 원리

이 사이트는 **백엔드 서버가 없습니다.** 자료 목록은 GitHub의 공개 Contents API (`https://api.github.com/repos/{owner}/{repo}/contents/...`)로 실시간 조회합니다. 따라서 Supabase, Firebase 같은 외부 서비스에 의존하지 않으며, 한 번 만들어두면 영구적으로 동작합니다.

브라우저 캐시(5분 TTL)와 GitHub의 강력한 CDN 덕분에 응답 속도도 빠릅니다.
