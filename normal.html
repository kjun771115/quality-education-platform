서로 다른 척도를 하나의 공통 언어로 — Z-점수를 통한 데이터 비교의 통계적 기반을 깊이 탐구합니다.

Z = (X − μ) / σ
스크롤
Section 01

표준화란 무엇인가?

표준화(Standardization)는 서로 다른 단위·척도를 가진 데이터를 평균 0, 표준편차 1의 공통 척도로 변환하는 과정입니다. 이를 통해 비교 불가능했던 데이터 간의 상대적 위치를 정확히 파악할 수 있습니다.

📐 표준화 (Standardization)

원본 데이터 X를 평균과 표준편차를 사용해 Z-점수로 변환합니다. 데이터의 형태는 유지하면서 척도만 바꿉니다.

Z = (X − μ) / σ

🔵 표준정규분포

평균 μ=0, 표준편차 σ=1인 특수한 정규분포입니다. 표기: Z ~ N(0, 1)

f(z) = (1/√2π) · e^(-z²/2)

📍 Z-점수의 의미

관측값이 평균으로부터 표준편차의 몇 배만큼 떨어져 있는지를 나타내는 무차원 수치입니다.

Z = 0 → 평균 위치
Z = ±1 → 1σ 거리

🎯 왜 표준화가 필요한가?

키(cm)와 몸무게(kg)처럼 단위가 다른 변수를 직접 비교할 수 없습니다. 표준화로 단위를 제거하여 공정한 비교가 가능해집니다.

단위 제거 → 상대적 위치 파악
📈 차트 ① — 정규분포 → 표준정규분포 변환 시각화

서로 다른 μ와 σ를 가진 정규분포들이 표준화 후 하나의 표준정규분포 N(0,1)로 수렴하는 과정을 보여줍니다.

【변환 전】 다양한 μ, σ의 정규분포

【변환 후】 표준정규분포 N(0,1) 수렴

Section 05

누적분포함수 (CDF)

표준정규분포의 누적분포함수 Φ(z)는 Z ≤ z인 확률을 나타냅니다. Z-표를 그래프로 표현한 것입니다.

📐 차트 ⑥ — 누적분포함수 Φ(z) 완전 시각화

하위 2.28%

Φ(+1) = 84.13%

🎓 차트 ⑦ — 수능 국어·수학 성적 Z-점수 분포 비교

2024학년도 수능 국어(μ=63.1, σ=12.7)와 수학(μ=55.2, σ=21.3) 성적을 표준화하여 난이도를 비교합니다.

📌 핵심 인사이트: 수학 σ=21.3은 국어 σ=12.7보다 훨씬 크므로 수학의 상위권과 하위권 격차가 극심합니다. 표준화 후 Z=+1.5 지점의 원점수는 국어 82.2점, 수학 87.2점으로 다릅니다. 동일한 Z-점수라도 과목마다 다른 원점수가 필요하다는 사실이 표준화의 핵심 가치입니다.
